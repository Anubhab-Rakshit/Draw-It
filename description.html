This is mainly a website for people who love to drawing and also to write notes


There are some steps to use the canvas

Some important points


  • The default color is red , to change the colour click on the color change tab beside the pen button
  • You cannot directly draw on the canvas , you have to press the pen button on it at first
  • The canvas can only be saved in dark background color so it is preffered not to use black colour pen while drawing
  • Though you do in light mode , it will be saved in black background only
